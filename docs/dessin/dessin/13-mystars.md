# デッサン（dessin）

## My天体メニュー
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------


### 目的:
- 天体検索で登録された天体の詳細を表示するために存在し、複数登録できることで、My観測点メニューとMy目的点メニューと併せて用いることで、My辻検索メニューを実現するものとする。
### 概要:
- 機能概要
  - 天体検索で登録された天体のIDと天体名と赤経赤緯で管理されるMy天体の詳細を表示天体と同じように表示する。
  - IDは、既定の表示天体は、固有名であるが、My天体は、連続する数字とする。
    - IDの採番は、連番の空きのある最小値から使用する。
      - 例:1,2,4,6が採番されている場合は、次に採番するのは、3である。
  - CSV入力とCSV出力で、ファイルによる登録ができる。
- デザイン概要
  - 表示天体メニューの下に配置する。
  - 0段目:
    - 1. 「My天体」ラベル:
      - 太字、左寄せ
    - 2. 「▲」ラベル:
      - 右寄せ
    - 共通事項:
      - タップすると、My天体メニューが開閉する。
      - 背景グレー、不透明
      - 初期状態:開いている。
  - 1〜N段目:My天体の件数分繰り返す
    - 凡例:0️⃣:段抜き、1️⃣:1行目、2️⃣:2行目、以下同様
    - 表示天体メニューでラジオボタン以外同じ
    - 0️⃣ラジオボタン
      - 削除対象の選択を表す。
    - 0️⃣チェックボックス
      - 辻ラインの表示/非表示のための選択用
    - 0️⃣カラーパレットボタン
      - My天体は、初期値を全て「薄紫、破線」とする。
      - 現在選択中の線色と線種(実線/破線)を表示して、クリックすると、スタイル設定小ウィンドウが開く。
        - スタイル設定小ウィンドウ
          - 赤、桃、橙、黄、黄緑、緑、水、青、藍、紫、薄紫、茶、焦茶、白、黒の全15色の線色切り替えボタンがある。
          - 実線、破線の選択ボタンがある。
          - リセットボタンで、既定線色/線種に戻せる。
          - 閉じるボタン
    - 1️⃣天体名 (太字、大きめ)
    - 2️⃣ID: 1 (灰色、小さめ)
    - 3️⃣赤経 0.1652h / 赤緯 1.0735° (灰色、小さめ)
    - 4️⃣出時刻 05:41 / 南中時 11:47 / 入時刻 17:54 (灰色、小さめ)
    - 5️⃣方位角 264.62° / 視高度 9.27° / 視半径 0.267° (灰色、小さめ)
  - N+1段目
    - 3. 「▲上へ」ボタン:
      - ラジオボタンで選択中のMy天体を1行上へリスト内の表示の位置を移動する。
    - 4. 「▼下へ」ボタン:
      - ラジオボタンで選択中のMy天体を1行下へリスト内の表示の位置を移動する。
  - N+2段目:
    - 5. 「削除」ボタン:
      - ラジオボタンで選択されたIDのMy天体をリストから削除するボタン。
      - このボタンを押下すると、実行する前に、確認メッセージ「My天体リストのの天体(ID:1, My天体名)を削除しますか？」（window.confirmで、OK、キャンセル）が表示される。
      - ローカルストレージからも削除される。
  - N+3段目:
    - 6. 「CSV入力」ボタン:
      - CSVファイルからMy天体のリストを入力できる。
      - このボタンを押下すると、実行する前に、確認メッセージ「My天体リストにCSVファイルから全て上書き入力・登録しますか？」（window.confirmで、OK、キャンセル）が表示される。
        - 確認メッセージの後にファイル選択ダイアログが開いて、デバイス内のファイルを選択できる。
      - CSVファイルから入力されると現在のリスト全て削除して、CSVファイルの内容で全て上書きをして、appState、ローカルストレージにも登録される。
      - CSVの見出し行は、1列目:天体ID、2列目:天体名、3列目:赤経、4列目:赤緯として、見出し行が存在する。
      - 制約:天体IDの数字の重複チェックをする。
      - 制約:CSVで登録できる件数の上限は1000件まで。CSVの見出し行を入れて1001行まで。
    - 7. 「CSV出力」ボタン:
      - My天体のリストをCSV出力できる。
      - このボタンを押下すると、実行する前に、確認メッセージ「My天体リストの登録内容をCSVファイルに出力しますか？」（window.confirmで、OK、キャンセル）が表示される。
        - 確認メッセージの後にブラウザからCSVファイルをダウンロードできる。
- 共通機能/デザイン
  - 背景色:
    - 1〜N+2段目は、背景白で、少し透明で、背景の地図が見える。
- コントロールの初期状態
  - ローカルストレージのMy天体リストの登録内容によって、登録されている内容を各My天体ごとに、2段目以降に行を追加して、コントロールに読み込んで表示する。
    - 読み込みに時間がかかる場合もあるので、非同期処理として実装する。
      - 読み込み中は、進行中のクルクルを表示する。絵文字の時計のアイコンの針がくるくる回転する。
  - ローカルストレージのMy天体リストがない場合は、0件（2〜N段目がない状態）になる。
### 仕様詳細：
- CSV処理
  - CSV入力時、名称以外の項目の値で、全角英数字記号は半角英数字記号に変換する。
  - ファイルの文字コードと改行コードは、MS Excel,Google SpredSheet,Windows,Mac,Android,iPhoneで互換性のあるものとする。
