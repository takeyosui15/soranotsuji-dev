# デッサン（dessin）
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
## 表示天体メニュー ( /docs/dessin/dessin/09-bodies.md を参照のこと )
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------


### 目的:
- 既定の天体の各種情報表示と、辻ラインの表示/非表示と辻ラインの線種/線色を切り替えられるようにする。
### 概要:
- 機能概要
  - 各既定の表示天体
    - 太陽
      - 既定表示、線色:赤、線種:実線
    - 月
      - 既定表示、線色:黄、線種:実線
    - 水星
      - 既定非表示、線色:水、線種:実線
    - 金星
      - 既定非表示、線色:桃、線種:実線
    - 火星
      - 既定非表示、線色:橙、線種:実線
    - 木星
      - 既定非表示、線色:赤、線種:実線
    - 土星
      - 既定非表示、線色:緑、線種:実線
    - 天王星
      - 既定非表示、線色:黄緑、線種:実線
    - 海王星
      - 既定非表示、線色:藍、線種:実線
    - 冥王星
      - 既定非表示、線色:紫、線種:実線
    - 北極星
      - 既定非表示、線色:黒、線種:実線
    - 北斗七星メラク
      - 既定非表示、線色:焦茶、線種:実線
    - オリオン座ミンタカ
      - 既定非表示、線色:白、線種:実線
    - すばる
      - 既定非表示、線色:青、線種:実線
    - オリオン大星雲M42
      - 既定非表示、線色:薄紫、線種:実線
    - こと座ベガ
      - 既定非表示、線色:橙、線種:破線
    - わし座アルタイル
      - 既定非表示、線色:緑、線種:破線
    - はくちょう座デネブ
      - 既定非表示、線色:黄、線種:破線
    - オリオン座ベテルギウス
      - 既定非表示、線色:赤、線種:破線
    - おおいぬ座シリウス
      - 既定非表示、線色:水、線種:破線
    - こいぬ座プロキオン
      - 既定非表示、線色:黄緑、線種:破線
  - 各既定の表示天体毎に以下の機能を表示する。
    - 凡例:0️⃣:段抜き、1️⃣:1行目、2️⃣:2行目、以下同様
    - 0️⃣チェックボックス
      - 辻ラインの表示/非表示のための選択用
    - 0️⃣カラーパレットボタン
      - 現在選択中の線色と線種(実線/破線)を表示して、クリックすると、スタイル設定小ウィンドウが開く。
        - スタイル設定小ウィンドウ
          - 赤、桃、橙、黄、黄緑、緑、水、青、藍、紫、薄紫、茶、焦茶、白、黒の全15色の線色切り替えボタンがある。
          - 実線、破線の選択ボタンがある。
          - リセットボタンで、既定線色/線種に戻せる。
          - 閉じるボタン
    - 1️⃣天体名 (太字、大きめ)
    - 2️⃣ID: Sun (灰色、小さめ)
    - 3️⃣赤経 0.1652h / 赤緯 1.0735° (灰色、小さめ)
    - 4️⃣出時刻 05:41 / 南中時 11:47 / 入時刻 17:54 (灰色、小さめ)
      - 出時刻、南中時、入時刻は、太陽系惑星以外の恒星は、算出時間がかかるため、非同期処理にして、計算待ち時は「--:--」を表示する。
    - 5️⃣方位角 264.62° / 視高度 9.27° / 視半径 0.267° (灰色、小さめ)
