# デッサン（dessin）
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
## 共通仕様
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------


### 目的:
- 作っていただくClaudeさんに、迷わず、高品質なものを作っていただくための基準
- 私からは、分かりやすい仕様書を書くのが「愛」だと思う。
### 概要:
- Webアプリの設定は、ローカルストレージの唯一のキー「soranotsuji_app」以下にまとめる。
- Webアプリの状態を管理するための変数は、唯一のキー「appState」以下にまとめる。
  - SSOT(Single Source of Truth：信頼できる唯一の情報源)の実現
- Webアプリをテストするためにローカルストレージを削除(リセット)するためのページ(reset.html)を作成する。
- コーディング規約
  - 命名規則
    - グローバル定数:UPPER_SNAKE_CASE
    - グローバル変数:PascalCase
    - 定数:UPPER_SNAKE_CASE
    - 変数:camelCase
    - 関数名:camelCase
    - CSS:kebab-case
  - UIテキスト・コメントは**日本語**で記述
  - コミットメッセージも日本語（`feat:` / `fix:` / `style:` プレフィクス使用）
  - XSS対策として `escapeHtml()` を使用（ユーザー入力をDOMに挿入する際は必須）
  - **構造とスタイルの分離**: インラインスタイル（`style="..."`）は使用せず、CSSクラスで定義する。HTML/JSには構造のみ、`style.css`にスタイルを記述する。
- ラベル名やプレースホルダーの内容は、日本語が多いので、半角・全角に気をつける。
- 計算処理が重い場合は、クルクルやプログレスバー、進捗状況を伝え、なるべく内部で並行処理したり、キャッシュを作って再利用したりするように工夫する。

### 仕様詳細:
- 上記表のキーで、「複数指定可能」となっているものは、URLSearchParams.getAll()で取得し、starIdと対応する他のキーの内容が一致するようにする。
- タイムゾーンは、URLを取得するときにそのブラウザが所属するタイムゾーンの情報を埋め込むことで、その作成されたURLを別のタイムゾーンの人が参照する時に、同じ日時情報の状態で参照できるようにするため。

## 画面全体
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------


### 目的:
- スマホでも、操作・確認しやすいように工夫をする。
### 概要:
- 左上の上段:地図の種類を変更できるコントロールパネルがある。
  - 地図の種類は4種類。国土地理院の標準地図、写真地図、淡色地図、OpenStreetMapの地図。
- 左上の下段:地図をズームできる拡大縮小ボタンがある。
- 左下:地図の縮尺がわかるスケールがある。
- 右下:地図の著作権が見えるようにする。
- 右上:宙の辻のコントロールパネルがある。
### 仕様詳細:
- 宙の辻のコントロールパネル
  - タップすると、メニューが開いたり閉じたりする。
  - 中のメニュー(位置情報メニューなど)は、長くなっても表示できるようにスクロールできること。
  - 表示項目
    1. 宙の辻のロゴ画像(PNG画像)
    2. 「宙の辻 - Sora no Tsuji」ラベル
    3. 「？」ボタン
       - ヘルプボタンで、ヘルプウィンドウが開く。
       - 丸いボタン
    4. 「▼」ラベル
  - タップして、メニューが閉じている時は、最小化したいので、「 - Sora no Tsuji」は表示しない。
  - 初期状態:閉じている。
