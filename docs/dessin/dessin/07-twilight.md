# デッサン（dessin）
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
## 薄明メニュー
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------


### 目的:
- 簡単に各種薄明の時刻と視高度を知り、ジャンプできるようにする。
### 概要:
- 機能概要
  - 夜明
    - 天文薄明\[始\]
      - -18°
    - 航海薄明\[始\]
      - -12°
    - 夜明
      - -7°21′40″
    - 常用薄明\[始\]
      - -6°
    - 日の出
      - 算出する
  - 日暮
    - 日の入
      - 算出する
    - 常用薄明\[終\]
      - -6°
    - 日暮
      - -7°21′40″
    - 航海薄明\[終\]
      - -12°
    - 天文薄明\[終\]
      - -18°
