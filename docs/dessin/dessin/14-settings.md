# デッサン（dessin）
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
## 設定メニュー
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------


### 目的:
- 気差係数を設定し、気差係数の有効/無効を設定できるものとする。
### 概要:
- 機能概要
  - 気差係数の有効/無効の変更
  - 気差係数を気圧/気温/気温減率から算出できる。
  - リセットボタンと登録ボタンがある。
  - 気差係数が無効の時は、読み取り専用になる。
- デザイン概要
  - My天体メニューの下に配置する。
  - 0段目:
    - 1. 「設定」ラベル:
      - 太字、左寄せ
    - 2. 「▼」ラベル:
      - 右寄せ
    - 共通事項:
      - タップすると、設定メニューが開閉する。
      - 背景グレー、不透明
      - 初期状態:閉じている。
  - 1段目:
    - 3. チェックボックス:
      - 「気圧」「気温」「気温減率」のテキストボックスの読み取り専用にする/しないを切り替えて、気差係数を全ての処理で有効/無効を切り替える。
    - 4. 「気差係数」テキストボックス:
      - 常に読み取り専用。
  - 2段目:
    - 5. 「気象条件からKを算出」ラベル:
  - 3段目:
    - 6. 「気圧」テキストボックス:
  - 4段目:
    - 7. 「気温」テキストボックス:
  - 5段目:
    - 8. 「気温減率」テキストボックス:
  - 6段目:
    - 9. リセットボタン:
  - 7段目:
    - 10. 登録ボタン:
- 共通機能/デザイン
  - 背景色:
    - 1段目は、背景白で、少し透明で、背景の地図が見える。
    - 2〜6段目は、グループにして、薄い青色の背景色にする。
