# デッサン（dessin）
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------
## バックアップメニュー
-------------------------------------------------------------------------------
-------------------------------------------------------------------------------


### 目的:
- Homeボタン、推し山ボタン、表示天体、My天体、My観測点、My目的点、My辻検索、設定のリストの一括バックアップをすることで、データの保護と、デバイス間のデータの同期を可能とするものとする。
### 概要:
- 機能概要
  - JSON形式のファイルで、一括バップアップとインポートができる。
  - appState、ローカルストレージとの整合性が保てる。
- デザイン概要
  - 設定メニューの下に配置する。
  - 0段目:
    - 1. 「バックアップ/インポート」ラベル:
      - 太字、左寄せ
    - 2. 「▲」ラベル:
      - 右寄せ
    - 共通事項:
      - タップすると、バックアップメニューが開閉する。
      - 背景グレー、不透明
      - 初期状態:開いている。
  - 1段目:
    - 3. 「バックアップ」ボタン:
      - このボタンを押下すると、実行する前に、確認メッセージ「Homeボタン、推し山ボタン、表示天体、My天体、My観測点、My目的点、My辻検索、設定のリストをバックアップファイルに全て出力しますか？」（window.confirmで、OK、キャンセル）が表示される。
        - 確認メッセージの後にブラウザからJSONファイルをダウンロードできる。
      - ファイル名にバックアップ実施日時を含める。
        - soranotsuji-バックアップ-YYYYMMDD-hhmmss.json
      - 以下の情報をJSON形式でバックアップする。
        - バックアップ実施日時
        - Homeボタン
          - 観測点緯度経度
          - 観測点標高
          - 観測点高
        - 推し山ボタン
          - 目的点緯度経度
          - 目的点標高
          - 目的点高
        - 表示天体
          - 天体ID
          - チェックボックスの状態
          - 天体色
          - 天体線種
        - My天体
          - My天体ID
          - My天体名
          - 赤経赤緯
          - チェックボックスの状態
          - 天体色
          - 天体線種
        - My観測点
          - 観測点ID
          - 観測点名
          - 観測点緯度経度
          - 観測点標高
          - 観測点高
          - メモ
        - My目的点
          - 目的点ID
          - 目的点名
          - 目的点緯度経度
          - 目的点標高
          - 目的点高
          - メモ
        - My辻検索
          - My辻検索ID
          - My辻検索名
          - 検索期間
          - 天体ID
          - 観測点ID
          - 目的点ID
          - 基準方位角
          - 基準視高度
          - オフセット方位角
          - オフセット視高度
          - 許容範囲方位角
          - 許容範囲視高度
          - フィルタ
          - 基準月齢
          - 許容範囲月齢
          - メモ
        - 設定
          - チェックボックスの状態
          - 気圧
          - 気温
          - 気温減率
    - 4. 「インポート」ボタン:
      - JSONファイルから一括インポートできる。
      - このボタンを押下すると、実行する前に、確認メッセージ「Homeボタン、推し山ボタン、表示天体、My天体、My観測点、My目的点、My辻検索、設定のリストをバックアップファイルから全て上書き入力・登録しますか？」（window.confirmで、OK、キャンセル）が表示される。
        - 確認メッセージの後にファイル選択ダイアログが開いて、デバイス内のファイルを選択できる。
      - appState、ローカルストレージと整合性が取れる。
      - My観測点、My目的点の地図上のマーカーも更新される。
- 共通機能/デザイン
  - 背景色:
    - 1段目は、背景白で、少し透明で、背景の地図が見える。
### 仕様詳細：
- JSON形式
  - ファイルの文字コードと改行コードは、MS Excel,Google SpredSheet,Windows,Mac,Android,iPhoneで互換性のあるものとする。
