# 要件定義: My天体セクション

セクションID: `sec-mystar`

## 目的

ユーザーが任意の天体（恒星・星雲・星団など）を赤経・赤緯で登録し、
他の天体と同様に方位角・高度・出入時刻を確認できるようにする。

---

## 機能一覧

### F5-1: My天体の登録

- 赤経（RA, 時）と赤緯（Dec, 度）をカンマ区切りで入力する
  - 例: `5.534,-0.299`（ミンタカ / オリオン座δ星）
- 「登録」ボタンで確定し、即座に計算に反映する
- 入力欄を空にして登録すると、デフォルト値（ミンタカ）にリセットする
- 形式エラー（カンマ区切りでない等）はアラートで通知する

### F5-2: 天体データの表示

- 以下の情報をリアルタイムで表示する:
  - 赤経 / 赤緯
  - 出時刻 / 南中時 / 入時刻
  - 方位角 / 視高度 / 視半径

### F5-3: 表示ON/OFF

- チェックボックスでMy天体の方位線の地図表示を切り替える
- OFF時もデータ表示は維持する

### F5-4: スタイル設定

- スタイルインジケーター（色と線種の表示）をクリックでパレットを開く
- 色と線種（実線/破線）を変更できる
- デフォルト: 薄紫（#DDA0DD）、実線

---

## デフォルト値

| 項目 | 値 | 備考 |
|---|---|---|
| 赤経 | 5.534h | ミンタカ（オリオン座δ星） |
| 赤緯 | -0.299° | 天の赤道近くの恒星 |

---

## 保存

- My天体の赤経・赤緯はLocalStorageに保存する
- 表示ON/OFF、色、線種もLocalStorageに保存する
