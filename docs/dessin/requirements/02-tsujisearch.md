# 要件定義: 辻検索セクション

セクションID: `sec-tsujisearch`

## 目的

特定の方位角・視高度に天体が来る日時を自動検索する。
「いつ・どこで天体が山と重なるか」を見つけるための中核機能。

---

## 機能一覧

### F2-1: 検索条件の設定

検索条件は6つのパラメータ＋検索期間で構成する:

| # | パラメータ | 説明 | デフォルト |
|---|---|---|---|
| ① | 基準方位角 | 目的点への方位角（自動計算） | 観測点→目的点の方位角 |
| ② | 基準視高度 | 目的点の視高度（自動計算） | 観測点→目的点の視高度 |
| ③ | オフセット方位角 | ①からのズレ量 | 0° |
| ④ | オフセット視高度 | ②からのズレ量 | 0° |
| ⑤ | 許容範囲方位角 | ①+③を中心とした許容幅 | 15° |
| ⑥ | 許容範囲視高度 | ②+④を中心とした許容幅 | 2.5° |
| — | 検索期間 | 検索する日数 | 365日 |

- ①②は観測点・目的点が変わると自動再計算される
- ③〜⑥はユーザーが手動で調整できる
- すべての値はLocalStorageに保存される

### F2-2: 検索の実行

- 「辻検索」ボタンで検索を開始する
- 表示中（チェック済み）の天体のみが検索対象となる
- 現在日時から未来方向に検索期間分を走査する
- 1分間隔で天体の方位角・視高度を計算し、条件に合致するかを判定する
- 各日で最も中心に近い結果（ベストマッチ）を1件記録する
- 天体1つあたり最大1461件（約4年分）で打ち切る
- 検索中はステータス表示を更新する（「検索中…」→件数表示）

### F2-3: 検索結果の表示

- 結果は下部パネルにテーブル形式で表示する
- 表示カラム: 天体名、日付（曜日付き）、時刻、月齢アイコン、方位角、視高度、判定
- 判定は3段階:
  - **◎**: 方位角・視高度ともに視半径以内（天体の中心が山頂に重なる）
  - **○**: 方位角・視高度ともに視半径×4以内
  - **△**: 条件範囲内だが上記に該当しない
- テーブルヘッダーをクリックするとソートできる（昇順⇔降順のトグル）
- 行をクリックするとその日時にジャンプし、地図上で確認できる
- 検索結果件数をフッターに表示する

### F2-4: 方位角の範囲判定

- 方位角の範囲判定は0°/360°をまたぐケースを正しく処理する
  - 例: 中心355°、許容15°の場合 → 340°〜10°が範囲内

---

## 画面構成

- 検索パラメータ入力はメインパネルのセクション内（初期状態: 折りたたみ）
- 検索結果は画面下部のオーバーレイパネルに表示
- 標高グラフと同時表示可能（パネル位置を自動調整）
