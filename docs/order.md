**CLAUDE.mdのファイル名変更について:**

**変更しない方がよいです。** `CLAUDE.md` はClaude Codeが自動的に認識する特別なファイル名です。プロジェクトルートに `CLAUDE.md` があると、セッション開始時に自動で読み込まれ、コーディング規約やプロジェクト情報をコンテキストとして使用します。

ファイル名を変更すると、Claude Codeでは自動読み込みが行われなくなります。

**他のエージェント（Antigravity等）との両立案:**

1. **`CLAUDE.md` はそのまま残す** — Claude Code用
2. **共通の指示ファイルを別途作成** — 例: `AGENTS.md` や `AI_INSTRUCTIONS.md`
3. **`CLAUDE.md` から共通ファイルを参照** — `CLAUDE.md` の冒頭に「詳細は `AGENTS.md` を参照」と記載

こうすれば、Claude Codeの自動認識を維持しつつ、他のエージェントでも同じ指示内容を共有できます。

Antigravity など他のエージェントは、どのようなファイル名を自動認識しますか？それによって最適な構成をご提案できます。

**問題1: 辻検索パネルが非表示のまま**
自動実行コードで `pnl.classList.remove('hidden')` や `tsujisearch-status` spanの作成を省略していたため、ボタンはアクティブになっても検索結果パネルが表示されませんでした。`toggleTsujiSearch()`と同等の処理（パネル表示・ステータスspan作成・`syncBottomPanels()`呼び出し）を追加しました。

**問題2: setNow()がURL復元日時を上書き**
