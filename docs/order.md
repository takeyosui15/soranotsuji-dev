### 回答 (2026-04-18) — 表示天体・My天体の表示順序変更

「出時刻 / 南中時 / 入時刻」の1行表示を、「出時刻 / 入時刻」と「南中時」の2行表示に変更しました。

#### 変更内容
- **HTMLテンプレート** (`renderCelestialList`, `renderMyStarsList`):
  - 既存の `riseset-{id}` span: 出時刻 / 入時刻のみ表示
  - 新規 `transit-{id}` span を追加: 南中時のみ表示
- **セッターロジック**:
  - 恒星・太陽系天体の両方で、`riseset-{id}` と `transit-{id}` を別々に更新
  - 非同期計算で南中時が遅れて反映される動作は維持
- **未設定時の表示**: `--:--` → `--:--:--` (他の時刻表示と統一)
- `searchStarRiseSet()` / `searchStarTransit()` の未設定フォールバック値も `--:--:--` に統一
