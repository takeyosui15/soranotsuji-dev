- 前回の修正でメモ比較を加えていましたが、デッサン仕様「辻検索IDと辻検索名の2つ以外が全て異なるもののみ追加」を「辻検索IDと辻検索名とメモの3つ以外が全て一致 → スキップ」の意味に解釈し直し、メモは除外しました。
- これで、辻検索IDと辻検索名とメモ以外の全ての内容が一致する場合は重複とみなしてスキップします。
- My観測点/My目的点の重複判定は緯度/経度/標高/高さのみを使用しており既にメモを含んでいないため、変更不要です。
