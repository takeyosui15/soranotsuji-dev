# PDCAサイクル記録

プロジェクトの改善サイクルを記録するドキュメントです。
各サイクルで「計画→実行→確認→改善」を回し、次のデッサンに活かします。

```
research.md  → モデルを見つめる（調査）
dessin.md    → デッサンする（構想メモ）
pdca.md      → 改善サイクルを回す（このファイル）
```

---

## サイクルの書き方

```markdown
## サイクルN: テーマ（日付）
### Plan — 何をどうしたいか
### Do — 何をやったか
### Check — 結果はどうだったか
### Action — 次にどう改善するか
```

- 全項目を埋める必要はありません。気づいたときに書く
- 小さなサイクルでOK。大きな改善より、回す頻度が大事
- CheckとActionが次のサイクルのPlanにつながる
