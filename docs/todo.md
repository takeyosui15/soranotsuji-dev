# TODOリスト

プロジェクトのやることリスト、気になること、AIからの提案を一元管理します。
このファイルを開くだけで、今やるべきこと・気にかかっていることが一覧できます。

## 運用ルール
- 完了したら `[x]` にチェックを入れる
- 不要になった項目は削除してOK
- 優先度が高いものを上に書く
- AIは作業中に気づいた改善案を「AIからの提案」に追記する
- バグを直したら、教訓を `knowledge/bugs.md` にも記録する
