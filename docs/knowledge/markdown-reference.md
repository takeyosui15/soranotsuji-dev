# Markdown 書式リファレンス

このファイルは、Markdownで使える書式をすべてサンプル付きで紹介します。
GitHubで表示する `.md` ファイルを書くときの参考にしてください。

---

## 見出し

```markdown
# 見出し1（h1）
## 見出し2（h2）
### 見出し3（h3）
#### 見出し4（h4）
##### 見出し5（h5）
###### 見出し6（h6）
```

# 見出し1（h1）
## 見出し2（h2）
### 見出し3（h3）
#### 見出し4（h4）
##### 見出し5（h5）
###### 見出し6（h6）

> **ポイント**: `#` の後にスペースを1つ入れること。

## リスト

### 箇条書き（順序なし）

```markdown
- 項目1
- 項目2
  - ネストした項目
  - もう一つ
    - さらにネスト
- 項目3
```

- 項目1
- 項目2
  - ネストした項目
  - もう一つ
    - さらにネスト
- 項目3

> `*` や `+` でも同じ結果になりますが、プロジェクト内では `-` に統一します。

### 番号付き（順序あり）

```markdown
1. 最初の手順
2. 次の手順
3. 最後の手順
```

1. 最初の手順
2. 次の手順
3. 最後の手順

### チェックリスト（タスクリスト）

```markdown
- [x] 完了した項目
- [x] これも完了
- [ ] まだ未完了
- [ ] これも未完了
```

- [x] 完了した項目
- [x] これも完了
- [ ] まだ未完了
- [ ] これも未完了

## テーブル（表）

```markdown
| 左揃え | 中央揃え | 右揃え |
|:---|:---:|---:|
| りんご | 100円 | 3個 |
| みかん | 80円 | 5個 |
| バナナ | 120円 | 1房 |
```

| 左揃え | 中央揃え | 右揃え |
|:---|:---:|---:|
| りんご | 100円 | 3個 |
| みかん | 80円 | 5個 |
| バナナ | 120円 | 1房 |

> **ポイント**:
> - `:---` = 左揃え（デフォルト）
> - `:---:` = 中央揃え
> - `---:` = 右揃え

## 水平線（区切り線）

```markdown
---
```

3つ以上のハイフン `---`、アスタリスク `***`、アンダースコア `___` で区切り線になります。
このプロジェクトでは `---` に統一します。

## エスケープ（特殊文字をそのまま表示）

Markdownの記号をそのまま表示したい場合は、前に `\` を付けます。

```markdown
\* アスタリスクをそのまま表示
\# シャープをそのまま表示
\- ハイフンをそのまま表示
\| パイプをそのまま表示
\` バッククォートをそのまま表示
```

\* アスタリスクをそのまま表示
\# シャープをそのまま表示
\- ハイフンをそのまま表示
\| パイプをそのまま表示
\` バッククォートをそのまま表示

## 目次の作り方

Markdownには自動目次機能はありませんが、セクションリンクを使って手動で作れます。

```markdown
## 目次

- [はじめに](#はじめに)
- [インストール](#インストール)
- [使い方](#使い方)
  - [基本操作](#基本操作)
  - [応用操作](#応用操作)
```

> **ルール**: 見出しのテキストを小文字にし、スペースを `-` に置き換える。
> 日本語の見出しはそのまま `#日本語の見出し` でリンクできます。

## このプロジェクトでの推奨スタイル

| 項目 | 推奨 | 避ける |
|---|---|---|
| 箇条書きの記号 | `-` | `*` `+` |
| 区切り線 | `---` | `***` `___` |
| 見出しの後の空行 | 入れる | 省略 |
| テーブルの区切り | `\|---\|` | `\|:---\|`（左揃えは省略可） |
| コードブロックの言語 | 必ず指定する | 省略 |
| 1行の長さ | 適度に改行 | 極端に長い1行 |
